This not finished yet, but it's a start.

## Getting Started

Stack used:

- Next.js
- Turbopack
- TailwindCSS
- TypeScript

First, run the development server:

```bash
pnpm intsall
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
